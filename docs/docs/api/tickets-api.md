# Tickets API

**Aim: Allow collaborative repository ticket management.**

Frontend can:
- Create, edit, and delete tickets.
- Mark tickets as “Resolved”.
- Assign users to tickets.
- Create tags, with an optional associated colour.
- Add and remove tags from tickets.
- Add tickets to a category (Kanban).
- Switch between “Kanban” and “Tabular” views.
- Filter tickets by tags.
